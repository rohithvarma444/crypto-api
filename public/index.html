Cryptocurrency API Documentation

Access real-time cryptocurrency data and statistical analysis

Get Latest Cryptocurrency Stats

GET/api/v1/stats?coin={symbol}

Returns the latest price, market cap, and 24-hour change for a specific cryptocurrency.

Query Parameters

coin (required) - Cryptocurrency symbol (bitcoin, ethereum, or matic-network)

Example Response

{
    "price": 45000.23,
    "market_cap": 850000000000,
    "percent_change_24h": 2.5
}

Get Price Standard Deviation

GET/api/v1/deviation?coin={symbol}

Calculates the standard deviation of prices from the last 100 records for a specific cryptocurrency.

Query Parameters

coin (required) - Cryptocurrency symbol (bitcoin, ethereum, or matic-network)

Example Response

{
    "deviation": 1234.56
}

Supported Cryptocurrencies

  • Bitcoin (bitcoin)
  • Ethereum (ethereum)
  • Polygon (matic-network)
